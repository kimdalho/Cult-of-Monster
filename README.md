# ✔ 게임스토리
- 당신은 고대의 성을 수호하는 괴물신으로 각종 괴물과 마법을 이용해 성을 방어해야 합니다. 외부의 세력으로부터 발생하는 위협은 계속해서 증가하고, 당신은 괴물신으로서 성을 수호해야 합니다. 성이 파괴되면 세계는 멸망하고 당신은 처음부터 다시 시작해야 합니다.
